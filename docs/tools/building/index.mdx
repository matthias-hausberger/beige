description: "Create custom tools and share them as multi-tool repositories"

A tool is a directory containing a manifest (`tool.json`), a handler (`index.ts`), and documentation. The gateway loads the handler and mounts the tool into agent sandboxes as an executable at `/tools/bin/<name>`.
